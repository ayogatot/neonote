## Features

### Core Functionality
- **Persistent Notes**: Double-click anywhere on a page to leave a note. It will be there when you come back.
- **Drag & Drop**: Easily move notes around the screen.
- **Minimize**: Minimize notes to tiny dots to avoid clutter.
- **Neubrutalist Design**: Bold borders, hard shadows, and pastel colors for a distinct look.

### Management
- **Dashboard**: View all notes on the current page in the popup.
- **Quick Navigation**: Click a note in the popup to scroll directly to it.
- **Clear All**: Easily remove all notes from a page.

## Installation

1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked" and select the `context-sticky` folder
5. The Context Sticky icon will appear in your toolbar

## Usage

### Creating Notes
1. Visit any website.
2. **Double-click** anywhere on the page.
3. A sticky note will appear. Type your note!

### Managing Notes
1. **Drag** the black header to move the note.
2. Click **_** to minimize the note to a dot.
3. Click **X** to delete the note.

### Popup Manager
1. Click the **Context Sticky icon** in your browser toolbar.
2. See a list of all notes on the current page.
3. Click a note card to scroll to it.

## Data Privacy

All data is stored locally in your Chrome browser using the `chrome.storage.local` API. No data is sent to external servers.
